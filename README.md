# HTML Portfolio

A clean and professional portfolio website built with pure HTML.

## Overview

This repository contains a personal portfolio website showcasing projects, skills, and experience. Built with semantic HTML, this portfolio provides a simple yet elegant way to present your work and accomplishments.

## Features

- **Clean Design**: Semantic HTML structure for easy navigation and accessibility
- **Responsive Layout**: Works across different devices and screen sizes
- **Easy to Customize**: Simple HTML structure makes it easy to modify and personalize
- **Fast Loading**: Lightweight HTML-only approach ensures quick page load times

## Getting Started

### Prerequisites

- A modern web browser
- A text editor (optional, for making modifications)

### Usage

1. Clone this repository:
   ```bash
   git clone https://github.com/LESLY16/html-portfolio.git
   ```

2. Open the HTML files in your web browser to view the portfolio

3. Customize the content to add your own information, projects, and links

## File Structure

```
html-portfolio/
├── index.html          # Main portfolio page
├── README.md          # This file
└── ...                # Additional HTML pages as needed
```

## Customization

To personalize your portfolio:

1. Edit the HTML files with your favorite text editor
2. Update personal information (name, bio, contact details)
3. Add your projects and accomplishments
4. Modify links to your social profiles or external sites
5. Customize styling by adding CSS (inline or external stylesheet)

## Deployment

Deploy your portfolio to GitHub Pages for free:

1. Go to your repository settings
2. Navigate to the **Pages** section
3. Select your branch (usually `main` or `master`)
4. Your site will be available at `https://LESLY16.github.io/html-portfolio/`
